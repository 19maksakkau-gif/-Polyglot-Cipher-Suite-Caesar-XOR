Polyglot Cipher Suite – Caesar & XOR in 7 Languages
Шифруй и дешифруй текст с помощью двух классических алгоритмов, реализованных на семи языках программирования.
Каждая версия обладает расширенным функционалом: автобрутфорс Цезаря, поддержка Unicode, работа с файлами, цветной вывод и не только.

🚀 Возможности
Шифр Цезаря – сдвиг букв (латиница + кириллица) с защитой от переполнения.

XOR-шифрование – побайтовый XOR с произвольным строковым ключом.

Брутфорс Цезаря – автоматический перебор всех сдвигов с выводом наиболее вероятного результата (по частотному анализу).

Ввод/вывод:

Из командной строки (аргумент -t / --text).

Из файла (-i / --input).

В файл (-o / --output).

Цветной вывод – результаты и ошибки подсвечиваются в терминале (ANSI).

Поддержка Unicode – корректно обрабатываются буквы любого алфавита (UTF-8).

Интерактивный режим (опционально) – если не переданы аргументы, программа запросит параметры.

📖 Использование
Общий синтаксис (для всех языков):

bash
<интерпретатор/исполняемый> [OPTIONS]
Доступные опции
Опция	Описание
-e, --encrypt	Режим шифрования (по умолчанию)
-d, --decrypt	Режим дешифрования
-c, --caesar	Использовать шифр Цезаря (с параметром сдвига)
-x, --xor	Использовать XOR (с параметром ключа)
-k, --key	Ключ: число для Цезаря, строка для XOR
-t, --text	Текст для обработки (если не указан, читается из stdin)
-i, --input	Входной файл
-o, --output	Выходной файл
-b, --bruteforce	Режим брутфорса (только для Цезаря)
-h, --help	Показать справку
Примеры
bash
# Шифрование Цезаря со сдвигом 3
python caesar_xor.py -e -c -k 3 -t "Hello World"

# Дешифровка XOR с ключом "secret"
python caesar_xor.py -d -x -k secret -t "1a2b3c"

# Брутфорс Цезаря (выведет все варианты и угадает лучший)
python caesar_xor.py -b -c -t "Lipps Asvph"

# Чтение из файла, запись в другой
python caesar_xor.py -e -x -k mykey -i input.txt -o output.txt
